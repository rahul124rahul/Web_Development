Arrays

Array

An array is just a list of values.

Index: Starts with 0.

Arrays are used for storing multiple values in a single variable.


Syntax & values

let arr = [1, true , 'Rahul' , 23.23 , 1 , {likes: '2K'}]

1. Use [] to create a new array, [] brackets enclose list of values.

2. Arrays can be saved to a variables.

3. Accessing values: Use [] with index.

4. Syntax Rules:

	- Brackets start and end the array.
	- Values separated by commas.
	- Can span multiple lines.

5. Array can hold any value, includeing arrays.

6. typeof operator on Array return Object.

Array Properties & Methods

1. Array.isArray() checs if a variable is an array.

2. Length property holds the size of array.

3. Common Methods:

	- push/pop : Add or remove to end.
	- shift/unshift : Add or remove from front.
	- splice : Add or remove element.
	- tostring : convert to string.
	- sort : Sort elements
	- valueOf : Get array itself.

4. Array also use reference like objects.

5. De-structuring also works for Arrays.
